# Sunrise Academy — Starter Website

This repository will contain a responsive static website starter for a school. It uses plain HTML, CSS, and a small JavaScript file for navigation and the contact form.

Quick setup:
1. The site files are in the repository root and `assets/`.
2. Open `index.html` in a browser to preview.
